## Overview

This project is a personal developer portfolio built to present my work in web development, UI engineering, and interactive digital systems.

It is designed as a **production-grade frontend system**, not a static portfolio page.

It demonstrates:

* Frontend engineering ability
* UI/UX system design
* Motion and interaction design
* Performance-first architecture

---

## Core Philosophy

This portfolio is built with one principle:

> Interfaces should feel alive, responsive, and intentional.

Every interaction is designed to respond to user behavior, not just display content.

### UI & Experience

## Project Structure

```text
src/
├── components/        # UI components
├── pages/             # Route-level views
├── hooks/             # Custom logic (scroll, motion, spatial effects)
├── lib/               # Utilities and API logic
├── styles/            # Global styles
```
